# Google Play Points Dashboard

A beautiful React dashboard to visualize your Google Play Points and Purchase History data.

## Features

- **Membership Overview**: View your current membership status, points balance, and level information
- **Points History**: Browse all your points transactions with filtering and search
- **Purchase History**: See all your purchases with detailed information and totals

## Getting Started

### Installation

1. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

Build for production:
```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Modern UI with gradients and animations

## Data Files

The app reads from two JSON files:
- `Play Points.json` - Contains membership and points history data
- `Purchase History.json` - Contains purchase history data

Make sure these files are in the root directory of the project.
